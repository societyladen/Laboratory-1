# Задание 1



## Задача 1



### Текст задачи

Сформировать список из длин вводимых строк.



### Алгоритм решения



Программа начинает свое выполнение с функции main. Вызывается функция lengthsLine для формирования списка длин строк, данная функция создает генератор списка и запускается цикл for от 1 до 5. В каждой итерации цикла выводится сообщение "Введите строку: ". Считывается строка, введенная пользователем, далее вычисляется длина этой строки и полученное значение добавляется в формируемый список. После завершения цикла сформированный список длин возвращается в функцию main. Полученный список выводится на экран и программа завершает работу.

# Задание 2



## Задача 1



### Текст задачи

Найти сумму чётных цифр натурального числа



### Алгоритм решения

Программа начинает свое выполнение с функции main. Выводится сообщение "Введите x для подсчета суммы четных цифр числа: ". Считывается целое число, введенное пользователем, и передается в рекурсивную функцию sumOfDigits. Функция sumOfDigits получает число и начинает его обработку. Проверяется условие: если число равно 0, функция возвращает 0 и рекурсия завершается. Если число не равно 0, то находится последняя цифра числа с помощью операции остатка от деления на 10. Проверяется, является ли эта цифра четной. Если цифра четная, она запоминается для сложения, если нечетная — запоминается 0. После этого функция вызывает саму себя для числа без последней цифры (результат целочисленного деления на 10). Запомненное значение складывается с результатом рекурсивного вызова. Рекурсия продолжается до тех пор, пока число не станет равным 0. Когда все цифры обработаны, функция возвращает общую сумму всех четных цифр. Полученная сумма возвращается в функцию main и выводится на экран. Программа завершает работу.

# Задание 3



## Задача 1


### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными

числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать

готовый класс Complex запрещено.



### Алгоритм решения



Программа начинает свое выполнение с функции main. Вызывается функция readComplex для ввода первого комплексного числа. Внутри функции readComplex выводится сообщение "Введи действительную часть: ", считывается число, затем выводится сообщение "Введи мнимую часть: ", считывается второе число. Введенные значения возвращаются в main как пара чисел и выводятся на экран функцией printComplex. Аналогично вызывается функция readComplex для ввода второго комплексного числа, которое также выводится на экран. После ввода чисел последовательно выполняются арифметические операции. Для сложения вызывается функция add, которая получает два комплексных числа, складывает их действительные и мнимые части и возвращает результат. Для вычитания вызывается функция subtract, которая вычитает части второго числа из первого. Для умножения вызывается функция multiply, которая вычисляет результат по формуле (ac - bd, ad + bc). Для деления вызывается функция divide, которая сначала вычисляет знаменатель c² + d², проверяет деление на ноль, затем умножает первое число на сопряженное ко второму и делит полученные значения на знаменатель. Каждый результат выводится на экран функцией printComplex. Для возведения в степень выводится запрос, какое число возвести (1 или 2). Считывается выбор пользователя, затем запрашивается степень. Вызывается рекурсивная функция pow с выбранным числом и степенью. Внутри функции pow проверяется: если степень равна 0, возвращается (1, 0); если степень равна 1, возвращается само число; иначе число умножается само на себя (степень раз) с помощью рекурсивных вызовов. Полученный результат выводится на экран. Программа завершает работу.
